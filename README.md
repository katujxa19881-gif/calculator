# Калькулятор экономики проекта

Интерактивный виджет для расчета себестоимости, рентабельности и целевой цены изделия. Оптимизирован для вставки на платформу Tilda.

## Основной функционал
* раздельный учет прямых и накладных расходов в нормо-секундах
* расчет экономики от целевой цены (таргета) за единицу изделия
* мгновенный пересчет маржи и прибыли при изменении скидки
* автоматическое сохранение всех введенных данных в память браузера
* отсутствие внешних зависимостей (чистый html, css и javascript)

## Как использовать в Tilda
1. создай новый блок т123 (html-код) на своей странице
2. скопируй всё содержимое файла `calculator-pro.html`
3. вставь код в блок и опубликуй страницу
